크레딧

Pokemon Card Pack Simulator

이 서비스는 포켓몬 TCG 팬 커뮤니티를 위한 비공식 팬 프로젝트입니다.

지적재산권

포켓몬(Pokemon) 및 관련 캐릭터, 카드 이미지, 상표는 주식회사 포켓몬(The Pokemon Company), 닌텐도(Nintendo), 크리처스(Creatures Inc.), 게임프리크(GAME FREAK inc.)의 지적재산입니다.

카드 이미지 출처: Pokemon TCG API (pokemontcg.io)

폰트

  • Noto Sans KR - Google Fonts (OFL License)
  • Outfit - Google Fonts (OFL License)

기술 스택

  • HTML5, CSS3, Vanilla JavaScript (ES6+)
  • Web Audio API (사운드 효과 생성)
  • PWA (Progressive Web App)
  • Vercel (호스팅)

서드파티 서비스

  • Adsterra (광고 네트워크)
  • Google Analytics 4 (사이트 분석)

오픈소스 라이선스

본 서비스는 외부 라이브러리 없이 순수 웹 기술로 구현되었습니다.

면책

본 서비스는 비영리 팬 프로젝트로서, 권리자의 요청이 있을 경우 즉시 중단됩니다.
